# Seat Reservations Project
This project is built in AngularJS

##API Server
To access the API, run the command:
```bash
node index.js
```
Within this same file, you can find the API code at:
``https://github.com/stepharriaza/Proyecto-Final-API-WEB.git`

##Project Compilation
To start the project, run the following command while the API is running:
```bash
http-server
```
Once the server is running, open your browser and go to `http://localhost:8080/index.html`. The application will automatically reload whenever code is modified in the source file.

##Features
In this project, you can perform the following functions:
- Register a user
- Make airline seat reservations
- Obtain reservation reports
